## 📌 Project Overview
This project is a fully integrated relational database pipeline built from scratch in Microsoft Excel. It tracks sales transactions, automates complex financial calculations, and delivers real-time visual insights through an interactive executive interface.

## 🛠️ Technical Features
* **Relational Data Layer:** Utilized `XLOOKUP` functions to dynamically link a live transaction ledger to a centralized product catalog database.
* **Automated Calculations:** Built a dynamic calculation layer computing Gross Revenue, Total Costs, Net Profit, and Profit Margins seamlessly across a scaling 15-row dataset.
* **KPI Alert System:** Programmed custom Conditional Formatting rules to instantly highlight high-margin rows (>50%) in crisp green for immediate risk and performance tracking.
* **Analytical Engine:** Structured a background `Pivot_Engine` matrix to aggregate, group, and process transactional records instantly.
* **Executive UI:** Designed a clean, gridline-free user dashboard featuring a professional navy blue header, custom currency formatting, dynamic bar charting, and interactive **Slicers** for on-demand filtering.
